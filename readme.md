Presented during the Facebook Singapore PHP User Group on 8 February 2017.

This repository contains a bare-bones Laravel 5.3 installation, with branches for each stage of the demonstration.

- start
- create_mailable
- create_view
- test_route
- subject_from_bcc
- image
- image_embedded
- register_controller
- queues
- plain_text
